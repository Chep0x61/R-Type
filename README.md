## Setup Project

#### Don't forget to install correctly needed dependencies for Conan to compile correclty.

<br />

```bash
mkdir build
```

### Then

```bash
./configure.sh
```

### Or

```bash
python3 -m pip install -r requirements.txt
```
```bash
cmake . -B build -D CMAKE_BUILD_TYPE=Debug
```
```bash
make -C build -j{number_of_threads}
```
